Important Concepts: Telling Computers What to Do

Lesson 1

Introduction to "serious" programming

Computer

A computer is an universal machine that can be programmed to do essentially any computation.

Computer Program

A computer program is a precise sequence of steps/links. Web browsers, games, mobile apps, and simple print statements are all examples of computer programs.

Programming Languages

A programming language is what programmers use to tell a computer what to do. Python, Perl, Ruby are examples of programming languages.

Python

Python is a programming language. It is an interpreter, which means it runs programs, interprets them, and then executes them. Python runs the programs in a language the computer can understand directly.

Grammar

Just like human languages have grammar, programming languages also do. A grammar is a specification of what is "correct" and what is "incorrect." In languages like English, people can often make sense of sentences that aren't technically "correct," due to experience and intuition. but computers do not have this capability. This means we have to write code that is exactly "correct" according to the Python interpreter, otherwise our code won't run.

Python Expressions

A Python "expression" is a legal Python statement. For example: print 1 + 1 is a valid expression, but print 1 + (without a number at the end) is not. This video does a good job of explaining the Python grammar for "expressions".

Lesson 2

Variables and Strings

What is a variable in Python?

Is method of using names to refer to a value. For example if sample_variable is a variable assigned with a value of "10", then the following command would display "10": print sample_variable.

What does it meant to assign a value to a variable? How is it done?

The code to assign a variable would look like the following:
sample_variable = 10.
The value of variables can be re-assigned.

What's the difference between what the equals sign = means in 2+3=5 and my_variable=5?

In 2+3=5, it stands for equals or 'same as' as in the arithmetic definition. In my_variable=5, it means "my_variable" is assigned the value of 5.

What are some ways that variables are useful?

Variables can be useful to programmers in many ways:

  • They improve code readability by using names that make sense to humans.
  • They give us a way to store the value of important data.
  • They give us a way to change the value of something (like in the line days = days-1
What is the difference between 2 + 2 and "2" + "2"?

In Python, 2 is a number while "2" is a string.

The code 2 + 2 would give 4.

The code "2" + "2" would give "22".

Lesson 3

Input -> Function -> Output

What is a function / procedure?

A function is something that accepts input does an action on it and produces output. For example a function named add_together might take 2 number values as input and output the total value when added together.

What is the difference between making and using a function?

When making a function you are defining it. The line of code for a function begins with def, giving the function a name, then the function parameters in parentheses. The parameters will have values that are assigned when the function is used or called.
Below is an example of a function:

def add_together(n):
total = n + n
return total
To use the function above, we can use the value of "2".
print add_together(2)
>>>4

How do functions help programmers avoid repetition?

A function is a tool that developers can write once and reuse/reference in other parts of the code. This prevents the programmer from having to write the same logic again if it is needed.

What happens if a function doesn't have a return statement?

The return keyword tells Python what to output. If the return is not used, the output of your function cannot be printed. Output will display as None

Lesson 5

Structured Data: Lists and For Loops

Lists

Lists are another method to store data in a more structured object. Nested Lists are lists, contained inside another list separated by another set of [].

For Loops

For Loops are a type of loop that iterates the same logic over on different parts of the structured data.

Mutation

Mutation is changing the value of a string after we've have already created/defined it as a variable. Redefining the variable mutates it to refer to the new value, however there is no way to refer to the original value.

Aliasing

Aliasing is two different ways to refer to the same object. For example two variables such as S and T could refer to the same object and if the value of S was changed, it would change the value of T also.

List Operations

The append operator is an example. It will add a new element to the end of a list, essentially mutating the list it is operating on.

<list>.append(<element>)

The plus operator is another one. It contatenates two lists together. It does not mutate the original list, it creates a new one.

<list> + <list>

The len operator len(<list>) provides the length or number of elements of a list it is passed to. It can also calculate the number of characters in a string.

The Index operator <list>.index(<value>) outputs the position of the value in the list.

For Loops

While loops are followed by a test expression. And if that test is true to execute the block. At the end of the block, go back to the while loop and repeat until test is False.

Lesson 6 - Stage 3

The Power of Abstraction

Module

A module is a file containing Python definitions and statements. This file can then be called inside your script mutliple times. This allows for flexiblity in your script and keeps it clean and orderly. This concept is similiar to how a CSS class can be used multiple times by an HTML script which would save some repeating lines of code.

Library

Python has a standard library of "built in" modules that provides access to the system resources that would otherwise be unavailable to Python. An example would be urllib that allows python to open a link to an Internet web address.

Class

A class is like a blueprint. A class can have both data and methods. It can be thought of as a basic receipe for a pie. Or in this case a receipe for code.

Instances

An instance is a separate example of a class. For example, the class Turtle indicates the face that it has size and a color. Individual instances of the class Turtle will contain information about their size and their colors as well. Using the pie receipe example, an instance could be a blueberry pie, or a rasberry pie.

Constructor

The constructor is the init method inside the class. It initializes all the data associated with the instance. It uses the keyword "Self"

Instance variable

Instance variables are all of the variables associated with a specific instance.

Instance methods

Instance methods are all of the functions inside the class that are associated with an instance and have the first arguement as self.

Inheritance

A subclass or Child that inherits attributes from the Parent class. This is another example of an advantage of Object Oriented Programming where it allows you to reuse code. Instead of copying the same piece of code many times over in a scipt, it can be referred to or inherited as a class.

Method overridding

The ability of a subclass to override a method that it inherited from its parent class.

Why OOP is a valuable tool for thinking about large projects.

The main concept of OOP is the object which contains data and subroutines. It has a 'state' of data and responds to subroutines. These subroutines can manipulate the data as the programmer sees fit. You start by identifiying the objects within a problem and the messages that these objects shouold respond to. The solution or the final program would be a set of objects each with it's own data set and responsibilties. The objects interact with sending messages to each other.

As it helps reducing duplicate code, it also helps in updating code. You only need to update the function once, and it's updated everywhere it's called. Large projects can also have a large set of data. Classes allow as many "data types" as you need. The data can be grouped in some manner which makes it more manageable.

Different from the standard top down approach of programming, OOP also includes a bottom up approach. There may be some parts of the project or issue to be solved that already have a known solution. These solutions are modules that can be reused and plugged into the code wherever they are needed. This allows programmers to reuse and share their work.

OOP provides flexibility and the ability to scale your code. It can be viewed as a collection of separate containers or objects that have actions between each other. The traditional top down methods are simply viewed as a group of functions and instructions.
